# ChromaVerse

Art makes the world a better place.

## About

ChromaVerse is a creative art gallery website featuring an outside-the-box design with vibrant, gender-neutral colors. The multi-page site showcases artistic works, provides information about the artist, and offers contact options.

## Pages

- **Home** (`index.html`): Welcome page with hero section and navigation
- **Gallery** (`gallery.html`): Collection of artworks with descriptions and editable price fields
- **About** (`about.html`): Information about the artist and creative philosophy
- **Contact** (`contact.html`): Contact form and studio information

## Features

- Responsive design with artistic gradients and animations
- Interactive gallery with hover effects and floating shapes
- Editable price fields for each artwork
- **Edit Mode**: Click "Edit Mode" button to customize content
  - Edit artwork titles and descriptions
  - Upload and change artwork images
  - Modify contact information (email, phone, address)
  - Changes are saved locally in your browser
- Smooth navigation between pages
- Creative typography with custom fonts
- Subtle background textures and glow effects
- Parallax scrolling effects

## Technologies Used

- HTML5
- CSS3 (with advanced features like gradients, animations, backdrop filters, and CSS Grid)
- JavaScript (for interactivity and dynamic effects)
- Google Fonts (Dancing Script and Roboto)

## How to View

Open any HTML file in your web browser to navigate the site. For a full experience, start with `index.html`.

## Note

Placeholder images are used in the gallery. Replace `placeholder1.jpg` through `placeholder6.jpg` with actual art images. Prices are set as editable inputs for demonstration purposes.
